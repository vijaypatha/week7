API: Application Protocal Interface

JSON: Data Format inspired from JS objects and are converted to String. 1) Root Object, 2) Keys are in quotes,3) simple datatypes. no methods or objects, and 4)no var =,

REST: is a protocol. Set of procedures or set of instructions. Protocol for sending and recieveing data

REST METHODS:

  • Get: get the data
  • Put : Modify. Change something that already exisits
  • Post: add to the API
  • Delete
